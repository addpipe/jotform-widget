This repository holds the HTML file used in the [Pipe Video Recorder Widget](https://eu.jotform.com/widgets/pipe-video-recorder-private-1) for Jotform. 

Alternatively, if you'd prefer not to use the plugin, follow this [tutorial](https://blog.addpipe.com/capturing-videos-with-pipe-in-a-jotform-contact-form/) for direct integration with Jotform.
