This repository holds the HTML file used in the Pipe Video Recorder Widget for Jotform. 

For more information, check out the [plugin](https://eu.jotform.com/widgets/pipe-video-recorder-private-1). Alternatively, if you'd prefer not to use the plugin, follow this [tutorial](https://blog.addpipe.com/capturing-videos-with-pipe-in-a-jotform-contact-form/) for direct form integration.
